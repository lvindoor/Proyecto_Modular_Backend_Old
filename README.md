### Prerequisites

```
* Java 8 u 11 OpenJdk 
* Maven
```

### Run Application

Para correr localmente la aplicación utilizar el siguiente comando de maven en la raiz del proyecto

```
mvn spring-boot:run
```

Para compilar la aplicación para producción utilizar el comando

```
mvn clean package
```
